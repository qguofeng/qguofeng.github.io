URL组成部分：schema://path?query
```bash
协议
路径
参数
```
例如
```bash
http://qiuguofeng.com
http://www.baidu.com
```
URL编码规则
```bash
除英文字母，数字和部分符号外，其他的全部使用百分好+十六进制值进行编码
```
url参数规则
```bash
参数以问号作为开始
参数对是key=value样式
参数对之间使用&号链接
```
